PS2 Guide

Un guide complet pour jailbreaker votre PlayStation 2.

Rejoignez Le Homebrew France pour obtenir de l'assistance ou nous aider à développer le guide.

N'oubliez pas de lire l'intégralité des pages avant de procéder pour ne pas prendre le risque de manquer une étape.

Informations avant de commencer

Cette manipulation sur la console a de multiples avantages :

  • Lire vos backups de jeux PlayStation 2 depuis une clé USB ou un disque dur. (plus aucun disque nécessaire)
  • Jouer à des consoles retrogaming à l'aide d'émulateurs.
  • Exécuter des "Homebrew" (logiciels non-approuvés par Sony) qui peuvent avoir toutes sortes d'utilités.
  • Jouer à des jeux qui ne sont pas normalement compatibles avec votre modèle de console (région différente)

Avertissement : Nous ne sommes pas responsables de l'utilisation que vous ferez de ce jailbreak PS2. Vous acceptez l'entière responsabilité de ce que vous faites.

I - Vérifier la compatibilité de sa console avec FreeDVDBoot

Dans un premier temps, il s'agira de vérifier le numéro de modèle de votre console.


		
Sur le modèle "FAT", le numéro de modèle se trouve à l'arrière, sur le modèle "Slim", il se trouve sous la console.

		

		
Prenez connaissance de votre numéro de modèle et gardez-le à portée de main

		
A présent, nous allons vérifier la version du lecteur DVD de votre console.

		
Pour ce faire :

		
    
			
  • Allumez votre console PlayStation 2.
    • 
			
  • Appuyez sur le bouton , un menu contenant plusieurs informations apparaîtra.
    • 
			
  • Votre numéro de version de lecteur DVD figure en bas de l'écran
    • 
		

			

			

			

			
A partir de là, vous avez les informations nécessaires pour savoir si votre console est compatible ou non avec la faille FreeDVDBoot.

			
Référez vous à ce tableau pour savoir si votre console est compatible :

			

			

			
Si votre numéro de modèle ainsi que votre version de lecteur DVD figurent sur ce tableau, votre PS2
		 est compatible avec la faille FreeDVDBoot

		

		
Si votre PS2 est compatible avec FreeDVDBoot, continuez vers la section FreeDVDBoot

		

			

		
Si votre PS2 n'est PAS compatible avec FreeDVDBoot, continuez vers la section FreeMCBoot
